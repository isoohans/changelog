Opinsys-järjestelmän versiot ja muutokset

Alle on listattu Opinsys-järjestelmän eri versiot ja niiden muutoslokit.



## Opinsys-järjestelmän versiot / Opinsys System versions

- Opinsys (fi) - Opinsys OS -käyttöjärjestelmän uusi tuotantoversio (Debian Buster)
- Opinsys (en) - Current production version of Opinsys OS
- Opinsys (de) - Aktuelle produktionsversion von Opinsys OS
- Opinsys - Opinsys OS -käyttöjärjestelmän vanhempi tuotantoversio (Debian Stretch)
- Lukiolaiskannettava (fi) - Opinsys OS -käyttöjärjestelmän lukiolaiskannettavaversio


## Arkisto / Archive
- Opinsys bootserver - Opinsys boottipalvelin
- Opinsys restricted - Käytöstä poistunut järjestelmäversio (Trusty)
- Opinsys extra - Kansainvälisillä asiakkailla käytössä ollut järjestelmä
- Lukiolaiskannettava - Opinsys Os:n lukiolaiskannettava versio (Debian Stretch)
